# Mon 5/4 — Squat 5s + Pull-up Retry + Carry

> Volume squat day (5s slot in the squat split). Heavy 3s land Fri 5/8. Carry slot lives here.

## 1. Back Squat — 5×5 Volume

```
Every 3:00 × 5 sets
5 reps @ 315#
```

+10# from 4/27's clean 305# 5×5 (RPE 8, all sets). 84% of 1RM, top of the volume zone. Belt and shoes on. This load is also the floor for Fri's hypertrophy back-offs.

**Decision rule:** set 1 RPE 8.5+ → drop to 310# for the rest. All 5 sets clear ≤ RPE 8 → next volume day try 320#.

## 2. Weighted Pull-up + Face Pull — superset

```
Every 2:30 × 4 rounds
A) Weighted pull-up: 5 reps @ 35#
B) Face pull (light cable): 15 reps, slow tempo, 1s hold at peak
```

**Pull-up:** retry of 4/27's 5/5/5/4. Goal today: clean 4×5 at the same load. Full hang to chin-over-bar, no kip. If set 4 fails again, hold 35# one more session before considering 40#.

**Face pull:** combined with Sat 5/9's sets = 8 weekly, clears the rear-delt deficit. High elbows, pull rope to forehead, hard scap squeeze. **Don't skip — this is the unsexy lift that holds heavy squat under the bar.**

## 3. Suitcase Carry

```
5 rounds, ~60s rest
50 ft right hand + 50 ft left hand @ 70–80# DB or KB
```

Heavy enough that the last 10 ft is hard. Stand tall, ribs stacked, hand opposite the load doesn't drift. Doubles as anti-lateral core. **Not KB swings** — those are still cycled out.
