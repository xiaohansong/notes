## 2. Face Pull + Copenhagen Plank — superset

```
4 sets, ~60s rest between rounds
A) Face pull: 15 reps, light cable, slow tempo, 1s hold at peak
B) Copenhagen plank: 25 sec each side
```

**Face pull:** Combined with Mon 5/4's superset face pulls = 8 weekly sets, clears the rear-delt deficit. Pulling to forehead, elbows high, scaps retracted. **Do not skip — same logic as Mon.**

**Copenhagen plank:** anti-lateral / adductor work. Side plank position with top leg's inner thigh on a bench, bottom leg hovering. Hold 25s each side, drop to 20s if hips sag. Different stimulus from Tue's Pallof — adductor + lateral chain rather than rotational. Skip the 4th round if either side is already shaky on round 3.

## 3. Weighted Pull-up — light touch

```
3 sets × 5 reps @ 25#  (NOT 35#)
```

Quality vertical-pull volume without re-loading lats hard the day after heavy squat. 25# (not your working 35#) keeps it as accessory, not a second hard pull session. If 5 reps feel RPE 6, fine — that's the point. RPE 8+ → stop the set and skip remaining sets. Save 35# attempts for fresh days (Mon).

## 4. Box Jumps — Power

```

3 reps onto 24" box
```

Step down between reps, don't rebound. Reset stance, full extension at top, soft landing. Stop early if any rep feels sloppy — power work is quality only. Last logged 4/17 at 24"×10 — keep height, fewer reps for crisper output. **Only planned power piece this week.**
