## 3. Bar Muscle-up Skill Block

```
3 rounds, ~90s rest between rounds
A) Explosive high pull-up: 3 reps — pull as high as possible (sternum > chin)
B) False grip hang: 15–20 sec
C) Jump-to-top negative or banded BMU: 2–3 reps (4s lower)
```

**Goal shift:** 35# weighted pull-up has stalled twice (4/27 → 5/4, both 5/5/5/4, same failure). Strict BMU is the new vertical-pull headline. Today is the first skill exposure — low volume, no load, focus on **height + transition**, not strength.

**A) Explosive high pull:** dead hang, pull fast, aim for the bar at sternum or lower. If you can't clear chest, prioritize speed over height — power up first, range comes later. Stop the round if a rep slows down; this is quality only.

**B) False grip hang:** wrists rolled over the bar so they sit on top, not in front. Feels weird and pinches at first — that's normal. 15s is fine on first exposure.

**C) Negative or banded BMU:** if you can jump or step to a support-hold on top of the bar, lower yourself under control (4s) to dead hang — that's the transition in reverse. If a thick band is available: loop over bar, foot in band, pull explosively + roll wrists over + press out. 2–3 reps total. Skip C if no setup works — A and B alone are enough today.

Rest between rounds is generous on purpose — this is skill/power, not metcon. Replaces the old "light touch 25#" slot.
